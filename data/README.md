## Dataset Information

This project uses the **Bank Marketing Dataset** from the UCI Machine Learning Repository.

Source:
UCI Machine Learning Repository - Bank Marketing Dataset

Moro, S., Rita, P., & Cortez, P. (2014). Bank Marketing [Dataset]. UCI Machine Learning Repository. https://doi.org/10.24432/C5K306.

The dataset contains approximately 41,000 observations from a direct marketing campaign conducted by a Portuguese banking institution.

## Data Access
The raw dataset is not included in this repository to ensure compliance with data licensing and repository best practices.

To reproduce the analysis:
1. Download the dataset from the official UCI repository.
2. Extract the file `bank-additional-full.csv`.
3. Place the file inside this `/data` directory.
4. Run the analysis notebooks sequentially.

This repository is intended for educational and analytical demonstration purposes only.
